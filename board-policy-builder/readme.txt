Comprehensive policy management system for Board of Education with builder interface, search, and PDF generation.

== Description ==
Complete policy management solution featuring drag-and-drop builder, PDF generation, search functionality, and taxonomy organization.

== Installation ==
1. Upload the plugin files to `/wp-content/plugins/board-policy-builder/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Policy Builder menu to start creating policies
